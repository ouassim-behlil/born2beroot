# Virtualization

## What is virtualization:

- **Virtualization** is a technology that creates _virtual envirements_ from from phisical hardware. In other words, it abstracts or simulates physical computing resources(CPU, storage, memory, network, etc). so they can be used more flexibly.[source](https://azure.microsoft.com/en-us/resources/cloud-computing-dictionary/what-is-virtualization)
